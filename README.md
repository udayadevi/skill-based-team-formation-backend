# Skill-Based Team Formation Platform (Backend)

## Overview

A backend application built using Node.js, Express.js, and MongoDB following MVC Architecture.

## Features

* User Registration
* User Listing
* Team Creation
* Team Listing
* Join Team Requests
* Accept/Reject Join Requests
* MongoDB Atlas Integration
* REST APIs
* Postman API Testing

## Tech Stack

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* Postman

## APIs

### Users

* POST /api/users
* GET /api/users

### Teams

* POST /api/teams
* GET /api/teams

### Join Requests

* POST /api/joinrequests
* GET /api/joinrequests
* PATCH /api/joinrequests/:id

## Project Structure

* Controllers
* Models
* Routes
* Config
* MVC Architecture
